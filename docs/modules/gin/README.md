&emsp;&emsp;Gin 是一个非常受欢迎的 Golang Web 框架，在 Github 上已经有 64k star，它和 Golang 的语法一样简洁明了，使得初学者得以迅速入门。

> 配置代理
```Go
GOPROXY=https://goproxy.cn,direct
```

## 下载 gin
```Go
go get -u github.com/gin-gonic/gin
```
